Plex Movie Selector User Guide

Introduction

This guide will explain how to use the Plex Movie Selector functionality in Home Assistant to select a random movie from your Plex library.

How to Use

To use the Plex Movie Selector, you will interact with the helpers you created in Home Assistant:

  • Genre: Select a genre from the Plex Movie Genre dropdown.
  • Decade: Select a decade from the Plex Movie Decade dropdown.
  • Director: Select a director from the Plex Movie Director dropdown.
  • Actor: Select an actor from the Plex Movie Actor dropdown.
  • Use Last Played Actors: Toggle the Plex Use Last Played Actors switch to enable/disable filtering by actors from your last played movie.
  • Collection: Select a collection from the Plex Movie Collection dropdown.
  • Content Rating: Select a content rating from the Plex Movie Content Rating dropdown.
  • Button: Press the Plex Movie Night Button to select and play a random movie based on your chosen criteria.

Example

Here is an example of how to select a random movie:

  1. Select "Action" from the Plex Movie Genre dropdown.
  2. Select "1980" from the Plex Movie Decade dropdown.
  3. Select "Steven Spielberg" from the Plex Movie Director dropdown.
  4. Select "Harrison Ford" from the Plex Movie Actor dropdown.
  5. Turn on the Plex Use Last Played Actors toggle.
  6. Select "Indiana Jones" from the Plex Movie Collection dropdown.
  7. Select "PG" from the Plex Movie Content Rating dropdown.
  8. Press the Plex Movie Night Button.

The system will then select a random, unwatched Action movie from the 1980s, directed by Steven Spielberg, starring Harrison Ford, from the Indiana Jones collection, with a content rating of PG, and also including at least one actor from the last movie you played. This movie will then start playing on your configured Plex client.

Creating Your HACS-Compliant Repository

Introduction

To make your Home Assistant project easily installable via HACS (Home Assistant Community Store), your Git repository needs to follow a specific structure. This guide will walk you through setting up your repository correctly.

Prerequisites

  • A Git client installed on your computer.
  • An account on a Git hosting service (e.g., GitHub, GitLab, Bitbucket).

Step 1: Initialize Your Git Repository

First, create a new empty directory on your computer for your project. Then, open your terminal or command prompt, navigate to that directory, and initialize a new Git repository:

mkdir plex-movie-selector-hacs
cd plex-movie-selector-hacs
git init

Step 2: Create the Required Directory Structure

HACS expects specific folders for AppDaemon apps and Python scripts. Create these directories within your new repository:

mkdir -p appdaemon/apps
mkdir -p python_scripts

Step 3: Place Your Project Files

Now, place the project files into their correct locations within your newly created repository structure:

  • Move plex_updater.py into appdaemon/apps/
  • Move apps.yaml into appdaemon/apps/
  • Move plex_movie_selector.py into python_scripts/
  • Place hacs.json at the root of your repository.
  • Place automations.yaml at the root of your repository.
  • Place configuration.yaml at the root of your repository.
  • Place scripts.yaml at the root of your repository.
  • Place deployment_guide.html at the root of your repository.
  • Place user_guide.html at the root of your repository.
  • Place GEMINI.md at the root of your repository.

Step 4: Add, Commit, and Push to Your Remote Repository

Once all files are in place, add them to your Git repository, commit the changes, and push them to your chosen Git hosting service (e.g., GitHub). Remember to create an empty repository on your hosting service first and link it.

git add .
git commit -m "Initial commit of HACS-compliant Plex Movie Selector"
git branch -M main
git remote add origin [YOUR_REMOTE_REPOSITORY_URL]
git push -u origin main

Replace [YOUR_REMOTE_REPOSITORY_URL] with the actual URL of your remote repository (e.g., https://github.com/your_username/plex-movie-selector.git).

Example Repository Structure

After following these steps, your repository structure should look like this:

your_repository_name/
├── hacs.json
├── appdaemon/
│ └── apps/
│ ├── plex_updater.py
│ └── apps.yaml
├── python_scripts/
│ └── plex_movie_selector.py
├── automations.yaml
├── configuration.yaml
├── scripts.yaml
├── deployment_guide.html
├── user_guide.html
└── GEMINI.md

Now your repository is ready to be added as a custom repository in HACS!
